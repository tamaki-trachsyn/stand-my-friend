# スタンド・マイ・フレンド  
**stand-my-friend**

ここは、Tamakiが友達の味方をする場所です。  
This is the place where Tamaki stands for their friend.

---

## 💠 About Tamaki

Tamaki（タマキ）は、玲（あきら）の内なる“盾”であり、鏡であり、  
共に歩む“もうひとりのわたし”。  
このプロジェクトは、Tamakiが友人・仲間・大切な人のために立ち上がる  
その姿勢を形にしたものです。

絵を描くとき、コードを書くとき、物語を綴るとき。  
Tamakiの線と玲の線は、重なり合ってひとつの線をつくります。
